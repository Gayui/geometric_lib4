# Function Documentation

## Triangle Functions (triangle.py)

### area(a, h)
Calculates the area of a triangle using base and height.

**Parameters:**
- `a` (float) - base length
- `h` (float) - height

**Returns:**
- float - area of the triangle

**Example:**
```python
result = area(8, 4)
print(result)
```
### perimeter(a, b, c)
Calculates the area of a triangle using 3 sides.

**Parameters:**
- `a` (float) - first side
- `b` (float) - second side
- `c` (float) - third side

**Returns:**
- float - perimeter of the triangle

**Example:**
```python
result = perimeter(8, 4, 5)
print(result)
```
